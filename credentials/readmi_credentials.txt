nstrucciones de Configuración de Credenciales - GCP
Para que este proyecto funcione correctamente, es necesario configurar las credenciales de Google Cloud Platform (GCP). Siga estos pasos:

1. Crear el Proyecto en GCP

• Inicie sesión en Google Cloud Console(https://console.cloud.google.com/).

• Cree un nuevo proyecto o seleccione uno existente.

• Habilite la API de BigQuery.

2. Generar el archivo .json de Credenciales

• Vaya a IAM y administración > Cuentas de servicio.

• Haga clic en Crear cuenta de servicio.

• Asigne los roles de `Administrador de BigQuery` y `Administrador de almacenamiento` (opcional).

• Una vez creada, entre a la cuenta, vaya a la pestaña Claves > Agregar clave > Crear clave nueva.

• Seleccione el formato JSON y descárguelo.

3. Ubicación del archivo

• Mueva el archivo `.json` descargado a la carpeta `credentials/` de este repositorio local.

• IMPORTANTE: El archivo debe estar en esta carpeta para que el volumen de Docker lo reconozca.

4. Modificar Conexiones en Airflow

Una vez levantado el entorno (`docker-compose up -d`), vaya a la interfaz de Airflow (`localhost:8080`):

• google_cloud_default:

  • Project Id: El ID de su proyecto de GCP.

  • Keyfile Path: `/opt/airflow/credentials/nombre-de-tu-archivo.json`.

• postgres_default:

  • Host: `postgres_oltp`.

  • Port: `5432`.

  • Login/Password: `Portafolio2026@`.
